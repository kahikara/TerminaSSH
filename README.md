# Termina SSH

A focused SSH client built for speed, clarity and daily use.

Termina SSH combines terminal access, quick connect, saved hosts, SSH tunnels, SFTP file browsing and a clean desktop UI into one lightweight app. It is designed to make common SSH workflows feel fast and polished instead of noisy and overcomplicated.

## Highlights

- Clean desktop UI
- Local Terminal support
- Quick Connect with password prompt
- Saved SSH connections
- Grouped connections
- SSH tunnels with start and stop controls
- SFTP browser with file actions
- Split terminal view
- Recent connections
- Active session overview
- Settings, themes and backups

## Features

### Terminal workflow

Termina SSH focuses on the core SSH experience.

You can open saved hosts, launch a Local Terminal, switch between active sessions and keep your workflow compact and organized.

### Quick Connect

Quick Connect is built for speed.

Enter username, host and port, confirm the password in a themed dialog and connect instantly without saving a host first.

### Saved connections

Store hosts for repeated use and keep them organized in groups.

This makes it easy to jump between servers without rebuilding the same connection setup every time.

### SSH tunnels

Termina SSH supports user friendly local SSH tunnels.

You can create, edit, delete, start and stop tunnels per server, copy local endpoints and see running state directly in the UI.

### SFTP browser

Browse remote files with an integrated SFTP panel.

Features include uploading, downloading, renaming, deleting, creating folders and editing remote files through a clean side panel workflow.

### Local Terminal

Local Terminal is treated like a first class entry in the app, so local shell work and remote SSH work live in the same interface.

### Settings and backup

The app includes settings for appearance and workflow, plus import and export support for configuration backups.
