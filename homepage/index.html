FAQ

A few practical answers

Is Termina SSH a web app?

No. It is designed as a desktop first experience for people who want a dedicated SSH manager without browser overhead.

Can I use existing key files?

Yes. PEM key support is part of the workflow so existing environments are easier to adopt.

Can backups move across systems?

That is the goal. A clean shared backup format makes Linux and macOS portability realistic and keeps future expansion easier.

Who is it for?

Developers, homelab users, sysadmins, and anyone who juggles multiple SSH connections and wants a cleaner workflow.
